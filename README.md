## Installation

- node.js v16.13.0
- yarn -v 1.22.17

## Getting Started

```
yarn install
yarn add vue@3.2.21
yarn add vite@2.6.14 --dev
yarn add @vue/compiler-sfc@3.0.7
yarn add @vitejs/plugin-vue@1.1.5 --dev
```
